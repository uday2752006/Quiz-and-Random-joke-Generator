# 🌐 Apex Planet Web Development Internship - Task 3

This repository contains the submission for **Task 3** of the Apex Planet Web Development Internship program. The focus of this task was to build a responsive and interactive web page using core frontend technologies.

## 📌 Task Objective

- Design and develop a responsive web layout.
- Implement interactive components using JavaScript.
- Ensure cross-browser compatibility and mobile responsiveness.
- Follow clean code and UI design practices.

## 🛠️ Technologies Used

- **HTML5** – For creating the structure of the web page.
- **CSS3** – For styling and responsive design.
- **JavaScript (ES6)** – For interactivity and dynamic behavior.
- **VS Code** – Development environment.
- **Chrome DevTools** – Debugging and layout testing.
